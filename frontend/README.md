# Insurance Claims Frontend

React + Material UI frontend shell for the Insurance Claims Management Portal.

## Current Purpose

This shell gives the MVP its first enterprise portal layout:

- Claimant/User role view
- Admin/Claims Reviewer role view
- Submit Claim screen placeholder
- My Claims screen placeholder
- Claim Details screen placeholder
- Admin Dashboard placeholder
- Claims Review Table placeholder
- Admin Claim Detail / Update Status placeholder

## Tech Stack

- React
- Vite
- Material UI
- JavaScript
- npm

## Run Locally

```bash
npm install
npm run dev
```

The dev server runs at:

```text
http://localhost:5173
```

The Spring Boot backend allows API requests from this origin during local development.
